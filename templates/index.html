Aplikasi analisis sentimen membantu mengidentifikasi pendapat masyarakat terhadap QRIS sehingga memberikan wawasan terkait penerapan QRIS di Indonesia. Proses analisis sentimen menggunakan metode Logistic Regression dan Metode Lexicon Based.

Alur Aplikasi
  1. Unggah Data: Mengunggah dataset yang berisi teks atau sentimen atau komentar atau pendapat mengenai QRIS.
  2. Praproses Data: Melakukan praproses data, seperti menghapus tanda baca, case folding, tokenization, menghapus stopwords dan stemming. menjadi kecil, dll.
  3. Pelatihan Model: Melatih model menggunakan algoritma machine learning Logistric Regression dan Metode Lexicon Based.
  4. Analisis Sentimen: Menganalisis sentimen menggunakan kedua metode.
  5. Lihat Hasil: Menampilkan hasil analisis sentimen.
